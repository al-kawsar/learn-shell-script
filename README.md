# Apa itu Shell ?

> Shell adalah sebuah program penterjemah yang berfungsi sebagai jembatan antara user dan kernel.

Shell adalah program antarmuka antara pengguna dan sistem operasi. Melalui shell, pengguna bisa memberikan perintah-perintah dalam bentuk teks untuk menjalankan program, mengelola file, dan melakukan tugas lainnya pada komputer. Contoh shell populer, diantarnya adalah : 

- Bourne shell(sh)
- C shell(csh)
- Korn shell(ksh)
- Bourne again shell(bash)
- dsb

Pada repository ini, saya menggunakan bash shell GNU yang merupakan pengembangan dari bourne shell.
